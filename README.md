1단계: 개발 환경 구축 (Section 1~5) → 필수. STM32CubeIDE 설치, 프로젝트 생성, 디버깅 설정부터 해야 코드가 돌아가요. (이 부분만 먼저 끝내고 넘어가면 OK)
2단계: GPIO - 핀 제어의 기초 (가장 중요 ⭐) → 완전 동의! 강의에서도 GPIO가 첫 번째 peripheral이고, bare-metal의 기본 근육이에요. 여기서 비트 조작 + 레지스터 이해가 몸에 배면 나머지 다 쉬워집니다.
3단계: Interrupts → 좋음. GPIO 다음에 인터럽트 배우면 "폴링 vs 인터럽트" 비교가 바로 가능해서 실전감 UP.
4단계: UART → 필수 ⭐. "말문 트기"라는 표현 딱 맞아요. UART로 로그 찍으면 디버깅이 10배 쉬워져서, 이후 I2C 실패할 때 "왜 안 돼?"를 로그로 바로 확인할 수 있음.
5단계: I2C - 최종 보스 MPU6050 → 목표 달성! 네가 산 센서가 I2C 기반이니, 여기까지 오면 "와, 진짜 성공했다!" 느낌 제대로 날 거예요.
